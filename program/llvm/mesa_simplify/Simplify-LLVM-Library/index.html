P1. 可被删除的库

1.1 具体架构相关的库

LLVMAMDGPUDisassembler
LLVMAMDGPUAsmParser
LLVMAMDGPUDesc
LLVMAMDGPUUtils
LLVMAMDGPUCodeGen
LLVMAMDGPUInfo

可以用 -DLLVM_TARGETS_TO_BUILD=""不要设置任何架构，这样就不会编译与架构相关的库。

1.2 汇编反汇编相关的库

LLVMAsmPrinter         # 输出汇编 
LLVMMCDisassembler     # 反汇编

1.3 指令选择相关的库

LLVMGlobalISel       # 新的全局指令选择方式
LLVMSelectionDAG     # 旧的指令选择

1.4 MIR相关的库

LLVMMIRParser    # LLVM中间过程读取MIR文件

1.5 可直接在LLVMBuild.txt和CMakeLists.txt中删除的库

LLVMObjectYAML  # 处理YAML格式的CodeView Debug Info
LLVMLineEditor

在llvm/lib/CMakeLists.txt和llvm/lib/LLVMBuild.txt中删除ObjectYAML,可以参考Remove ObjectYAML。
