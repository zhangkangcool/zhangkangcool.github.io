git 切换远程分支

git clone只能clone远程库的master分支，无法clone所有分支，解决办法如下：

git branch -a，列出所有分支名称如下：


remotes/origin/dev
remotes/origin/release


git checkout -b dev origin/dev，作用是checkout远程的dev分支，在本地起名为dev分支，并切换到本地的dev分支
