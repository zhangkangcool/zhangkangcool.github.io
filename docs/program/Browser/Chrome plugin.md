# 程序包无效: CRX_HEADER_INVALID"错误

- 将插件crx文件的后缀改成zip，然后再拖入到chrome浏览器的扩展应用中，即可添加成功
- 打开chrome浏览器，输入`chrome://extensions/`进入扩展应用页面；
- 将下载的插件crx文件直接拖拽到扩展应用页面，会出现确认添加插件的弹框，点击添加按钮即可完成插件的安装。
