



### 1. UNSUPPORTED

一般和OS相关，其个OS不支持

```shell
// UNSUPPORTED: system-windows
```







### 2. REQUIRES



```
// REQUIRES: system-linux
// REQUIRES: asserts
// REQUIRES: z3
```







### 3. XFAIL

```
// XFAIL: asserts
// XFAIL: *    // 所有的case都错误
// XFAIL: target=aarch64-pc-windows-{{.*}}
```
